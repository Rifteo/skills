# Security Policy

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities privately using [GitHub's Security Advisories](../../security).

We will acknowledge within 48 hours and aim to resolve within 14 days.

## Scope

**In scope:**
- Content injection risks that could cause an AI agent to execute unintended actions
- Malicious payloads embedded in skill files
- Dependency vulnerabilities with active exploits

**Out of scope:**
- Vulnerabilities in systems that skills describe how to test
- Issues with third-party tools referenced in skill steps
- Security issues in your own environment or agent setup
