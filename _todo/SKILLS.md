# Planned Skills

Skills flagged for development. Pick one up and open a PR.

## Bug Bounty - Hunting

| Skill | What it covers |
|---|---|
| `hunt-ato` | Account takeover - credential stuffing, session fixation, password reset flaws, OAuth token theft |
| `hunt-auth-bypass` | Authentication bypass - logic flaws, default credentials, forced browsing, token forgery |
| `hunt-mfa-bypass` | MFA bypass - code reuse, response manipulation, backup code abuse, SIM swap indicators |
| `hunt-oauth` | OAuth attacks - state parameter, redirect_uri bypass, token leakage, implicit flow abuse |
| `hunt-saml` | SAML attacks - signature wrapping, XXE in assertions, replay, open redirect in RelayState |
| `hunt-cache-poison` | Web cache poisoning - unkeyed headers, fat GET, cache deception |
| `hunt-http-smuggling` | HTTP request smuggling - CL.TE, TE.CL, TE.TE variants and exploit chains |
| `hunt-race-condition` | Race conditions - limit-once bypass, coupon abuse, concurrent state manipulation |
| `hunt-llm-ai` | LLM/AI feature testing - prompt injection, data exfiltration via AI, indirect injection |
| `hunt-web3` | Web3/Immunefi - reentrancy, flash loan manipulation, oracle abuse, access control, integer overflow |
| `vuln-chainer` | Given a confirmed finding, enumerate and test all companion bugs from the chain signal table |
